# Autonomia Digital:

Projeto de Extensão voltado para a concientização acerca fraudes digitais através de uma aplicação web educativa.

## Como acessar
- **Site Online (Firebase Hosting):** https://autonomia-digital-552db.web.app/
- **Código:** Este repositório contém o código-fonte desenvolvido.

## Tecnologias utilizadas
- HTML5, CSS3 e JavaScript (ES6 Modules)
- Firebase (Cloud Firestore e Hosting)

## Objetivo
Criar uma ferramenta interativa para alfabetização digital, permitindo a coleta de dados de desempenho para análise de vulnerabilidade social em Limeira/SP.
